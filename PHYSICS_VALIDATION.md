## 1. 方程与数值模型

自由刚体使用：

```text
m dv/dt = sum(F)
I domega/dt = sum(tau)
tau = r x F
Delta v = J/m
Delta omega = (r x J)/I
```

圆—圆和圆—地面接触使用法向/切向顺序冲量、库仑静/动摩擦与分离的位置修正。法向恢复满足：

```text
v_relative_after dot n = -e_effective (v_relative_before dot n)
e_effective = restitution / (1 + damping)
```

轨道采用弧长坐标 `s`。实心圆盘使用 `I = 1/2 mR^2`，无滑动条件为 `v = R omega`；斜坡无滑动加速度为 `a = 2/3 g sin(theta)`。单侧轨道所需法向力小于零时触发离轨。

变化场按：

```text
a = g + qE/m + q(v x B)/m + F_external/m
```

合成。自由世界使用 RK4，`k1` 至 `k4` 分别在自己的 `t, x, y, vx, vy` 中间状态求值；有限范围场在每个阶段重新判断是否生效。轨道约束使用二阶中点更新，并在步内求解端点到达时间。

## 4. 守恒、耗散与能量口径

- 无外力、无摩擦、`e=1` 的圆碰撞以总线动量和总动能零漂移作为门禁。
- `e<1`、正阻尼或动摩擦的能量下降来自冲量/摩擦模型，不对结果做事后能量修补。
- 多接触的 `dissipatedEnergy` 使用接触求解前后系统净动能差；逐接触中间归因另存，避免完全弹性三体冲量传播被误报为耗散。
- 变化场没有声明标量势函数时，势能和机械能明确为 `null`/不可用，不用场强预览值伪造势能。
- 持续外力或变化场作用于轨道时仍输出动能，但不声明势能或机械能守恒。

## 6. alpha 适用范围与限制

1. 碰撞窄相仅支持圆和显式正碰撞半径的质点；杆、圆环、矩形和任意形状会明确拒绝。
2. 轨道约束对象在 v1.5.0-alpha.1 中作为独立子系统求解，不与自由/接触对象发生碰撞。
3. `release/open` 端点或负法向力触发离轨时，结果停在离轨状态；剩余时长不接续二维自由飞行。
4. 折线拐角使用不做功的理想约束冲量；需要连续曲率时必须使用圆弧或 Bezier。
5. 当前磁场是垂直二维平面的 `Bz`，不是任意三维磁场。
6. 数值结果适用于教学、设计探索和软件验证；alpha 版本不替代安全关键工程复核、法规验算或试验认证。
