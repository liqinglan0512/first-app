# Computational Mechanics Solver v1.5.0-alpha.1 发布说明

发布日期候选：2026-07-12
状态：Alpha / 草稿 PR 候选，尚未部署、尚未打标签

## 主要更新

### 同步二维动力学世界

- 新增真实平动与转动状态、质心惯量、角度、角速度、角加速度和力矩。
- 支持质心、局部偏移和世界坐标作用点；偏心力与偏心冲量同时改变平动和转动。
- 支持圆—圆、圆—地面接触，恢复系数、静/动摩擦、阻尼、多接触迭代、穿透修正和自适应子步。
- 强外部加速度参与防穿透行程估计；超过计算上限时安全失败，不静默穿透。

### 轨道、斜坡与滚动

- 支持直线、倾斜直线、折线、圆弧和三次 Bezier 平滑轨道。
- 支持质点滑动、实心圆盘无滑动滚动、滑动转滚动、轨道摩擦、恢复、端点停止/反弹/释放与负法向力离轨。
- 输出到达时间、坡底状态、能量、滑动、滚动约束误差和离轨原因。

### 安全变化场

- 重力、电场和二维垂直磁场支持常量、时间、空间和时间—空间表达式。
- 自建受控 AST，只允许规定变量、运算、函数和常量；不执行工程中的 JavaScript 或 Python 代码。
- RK4 四阶段使用各自的中间时间、位置、速度和有限场范围。
- 无标量势函数的变化场明确把势能/机械能标为不可用。

### 工程、界面与报告

- 动力学工程升级到 `cms-dynamics-project@2`，保留 `@1` 自动迁移。
- 保存/打开/撤销保留接触材料、角状态、作用点、地面、轨道、约束和变化场。
- 工作台新增刚体/接触、`y=0` 地面、角初值、作用点和变化场控件。
- 报告新增角运动、接触、轨道和动态限制说明。
- 提供三个可直接打开的 v1.5 示例工程：碰撞、轨道滚动、变化场。

### 账户与权益（随集成分支交付）

- 服务器账户、Argon2id 密码、会话/CSRF/同源校验、限速和头像存储。
- Free、Plus、Pro、Internal Tester、Admin 服务端权益矩阵。
- Plus/Pro 当前只展示，不提供购买；PINN 仍为等待名单，不解锁求解器。
- Internal Tester 不继承 Admin；正式报告与高级导出由服务端鉴权。

## 验证结果

- Python 77/77；全部前端测试通过。
- v1.5 核心数值模块行/语句覆盖率 95.12%，函数覆盖率 95.80%。
- 弹性碰撞动量/动能零漂移；`e=0.5` 能量损失与理论值一致。
- 实心圆盘斜坡到达时间、坡底速度、滚动约束和能量达到数值舍入精度。
- 圆弧轨道步长减半误差比 0.2489，符合二阶收敛趋势。
- 宽屏、桌面、手机真实浏览器验收通过，控制台错误 0。

详细证据见 `PHYSICS_VALIDATION.md`、`SECURITY_REVIEW.md` 和 `release/v1.5.0-alpha.1/browser-qa.json`。

## 兼容性与迁移

- v1 动力学工程在打开时迁移为 v2；旧力/冲量默认作用于质心。
- v2 工程含有 v1 客户端不认识的轨道、接触和变化场字段，不保证向旧版反向兼容。
- 未实现碰撞几何现在 fail-closed；启用接触的杆、圆环、矩形和任意形状不会再被静默当作圆。

## 已知限制

- 轨道对象尚未与自由/接触对象进入同一碰撞世界。
- 离轨和开放端点停在事件状态，不继续剩余时长的自由飞行。
- 折线拐角是理想约束冲量，不代表真实圆角接触应力。
- 变化场未声明势函数时不提供势能/机械能守恒结论。
- 磁场仅支持二维平面法向 `Bz`。
- Plus/Pro 购买和 PINN 求解器尚未开放。
